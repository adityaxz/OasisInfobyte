# Password Generator Application

## 1. Objective

The objective of this project is to develop an advanced Password Generator application using Python. The application generates secure passwords based on user-selected criteria and provides password strength evaluation.

---

## 2. Project Description

A graphical user interface (GUI) was developed using Tkinter to allow users to generate secure passwords with customizable options.

Users can choose:

- Password length
- Uppercase letters
- Lowercase letters
- Numbers
- Symbols

The application evaluates password strength and supports clipboard integration for easy copying.

---

## 3. Technologies Used

- Python
- Tkinter
- Random Module
- String Module
- Pyperclip

---

## 4. Features Implemented

### Password Generation

The application generates random passwords using selected character categories.

### Character Selection

Users can include:

- Uppercase letters
- Lowercase letters
- Numbers
- Symbols

### Password Strength Analysis

Passwords are categorized as:

- Weak
- Medium
- Strong

based on complexity and length.

### Clipboard Integration

Generated passwords can be copied directly to the system clipboard.

---

## 5. User Interface

The GUI includes:

- Password length input
- Character type selection checkboxes
- Generate Password button
- Password display field
- Password strength indicator
- Copy Password button

---

## 6. Security Considerations

The application promotes secure password creation by:

- Supporting complex character combinations
- Encouraging longer passwords
- Evaluating password strength
- Preventing empty character selections

---

## 7. Results

The application successfully generates secure random passwords according to user preferences.

Password strength evaluation correctly classifies generated passwords, and clipboard functionality improves usability.
