# Weather Application

## 1. Objective

The objective of this project is to develop an advanced Weather Application using Python. The application retrieves real-time weather information for user-specified cities and displays important weather parameters through an interactive graphical interface.

---

## 2. Project Description

A graphical user interface (GUI) was developed using Tkinter to allow users to search for weather information of different cities.

The application displays:

* Temperature
* Humidity
* Wind Speed
* Weather Condition

The application also maintains a recent search history and retrieves live weather data using an online weather service.

---

## 3. Technologies Used

* Python
* Tkinter
* Requests Library
* wttr.in Weather Service

---

## 4. Features Implemented

### Live Weather Retrieval

The application fetches real-time weather information for the city entered by the user.

### Weather Information Display

The following information is displayed:

* Temperature
* Humidity
* Wind Speed
* Weather Condition

### Search History

Recent city searches are stored and displayed within the application.

### Error Handling

The application validates user input and handles errors during data retrieval.

---

## 5. User Interface

The GUI includes:

* City name input field
* Get Weather button
* Weather information display area
* Recent search history section
* User-friendly dark theme interface

---

## 6. Data Retrieval Process

The application retrieves weather information by:

* Accepting a city name from the user
* Sending a request to the weather service
* Processing the returned weather data
* Displaying formatted results in the GUI

---

## 7. Results

The application successfully retrieves and displays weather information for different cities.

Weather details such as temperature, humidity, wind speed, and weather conditions are presented clearly. Search history functionality improves user convenience by keeping track of recent city lookups.
