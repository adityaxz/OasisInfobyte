# BMI Calculator Application

## 1. Objective

The objective of this project is to develop an advanced Body Mass Index (BMI) Calculator using Python. The application calculates BMI based on user-provided weight and height, classifies health status, stores calculation history, and visualizes BMI trends.

---

## 2. Project Description

A graphical user interface (GUI) was developed using Tkinter to provide an interactive BMI calculation system.

The application allows users to:

* Enter weight and height
* Calculate BMI instantly
* View BMI category
* Store previous records
* Display BMI history
* Visualize BMI trends using graphs

---

## 3. Technologies Used

* Python
* Tkinter
* Pandas
* Matplotlib
* CSV File Storage

---

## 4. Features Implemented

### BMI Calculation

The application calculates BMI using the standard formula:

BMI = Weight (kg) / Height² (m²)

### BMI Classification

Users are categorized as:

* Underweight
* Normal Weight
* Overweight
* Obese

based on calculated BMI values.

### History Storage

Each BMI calculation is saved in a CSV file for future reference.

### History Viewer

Users can view previously recorded BMI calculations directly within the application.

### BMI Trend Visualization

A graphical representation of BMI history is generated using Matplotlib.

---

## 5. User Interface

The GUI includes:

* Weight input field
* Height input field
* Calculate BMI button
* View History button
* Show Graph button
* BMI result display
* BMI category display

---

## 6. Data Management

BMI records are stored in a CSV file named:

bmi_history.csv

Stored information includes:

* Weight
* Height
* BMI Value
* Category

---

## 7. Results

The application successfully calculates BMI values, categorizes users according to standard health ranges, stores historical data, and visualizes BMI trends through graphical analysis.

The GUI provides a user-friendly experience and supports basic health monitoring.
